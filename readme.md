# learn git

## config
```sh
git --config global user.name "xxx"
git --config global email.name "email"
```

## git start
```sh
mkdir learngit
cd learngit
git init
git add readme.md
git commit -m "wrote readme file"
git add file1.txt file2.txt
git commit -m "add 2 files"
```

## 时光机
```sh
git status
git diff # 看修改
```
